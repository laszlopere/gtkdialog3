<tree>

A multi-column list view with sortable column headers. Items are defined with pipe-separated values. The widget exports the value of one chosen column for the selected row.

Child Tags

  • <label> — pipe-separated column headers, e.g. Name|Size|Date
  • <item> — a row with pipe-separated values
  • <input> — shell command producing pipe-separated rows
  • <variable> — shell variable for the selected value

Attributes

  • exported-column — which column (0-based) to export (default: 0)
  • selection-mode — single, browse, multiple, or none
  • rules-hint — alternate row shading for readability

Example

<tree exported-column="0" selection-mode="browse">
  <label>Name|Description</label>
  <item>Alice|Developer</item>
  <item>Bob|Designer</item>
  <variable>TREE1</variable>
  <action signal="cursor-changed">echo $TREE1</action>
</tree>
