<webview>

An embedded web browser powered by WebKitGTK. It can display remote URLs, local HTML files, or HTML content generated by shell commands. The current URI is exported when the dialog closes.

Child Tags

  • <default> — initial URL or file:// path to load
  • <variable> — shell variable for the current URI
  • <input> — shell command whose output is loaded as HTML
  • <input file> — read a file; auto-detects URL, path, or HTML

Runtime Actions

  • set:var.uri=url — navigate to a new URL at runtime
  • refresh:var — re-run the <input> command

Example

<webview space-expand="true" space-fill="true">
  <variable>webPage</variable>
  <default>file:///tmp/home.html</default>
</webview>

<button>
  <label>Go</label>
  <action>set:webPage.uri=https://example.com</action>
</button>
