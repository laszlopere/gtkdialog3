<edit>

A multi-line text editor with scrollbars. It can be pre-filled from a file or shell command and its full contents are exported when the dialog closes. Suitable for notes, configuration editing, or log display.

Child Tags

  • <default> — initial text content
  • <variable> — shell variable for the full text
  • <input> — shell command whose output fills the editor
  • <input file> — load contents from a file path
  • <output file> — save contents to a file on exit
  • <action> — command to run on text changes

Attributes

  • editable — set to false for read-only display

Example

<edit>
  <input>cat /etc/hostname</input>
  <variable>edtNotes</variable>
  <height>200</height>
  <width>400</width>
</edit>
