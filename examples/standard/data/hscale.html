<hscale>

A horizontal slider that lets the user pick a numeric value by dragging a handle along a track. Useful for settings like volume, brightness, or any bounded numeric range.

Child Tags

  • <default> — initial numeric value
  • <variable> — shell variable for the current value
  • <input> — shell command whose output sets the value
  • <action> — command to run when the value changes

Attributes

  • range-min — minimum value (left end)
  • range-max — maximum value (right end)
  • range-step — step increment
  • range-value — initial position

Example

<hscale range-min="0" range-max="100"
        range-step="1" range-value="50">
  <variable>sclVolume</variable>
  <action>echo Volume: $sclVolume</action>
</hscale>
