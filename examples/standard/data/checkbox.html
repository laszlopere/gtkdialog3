<checkbox>

A toggle button with a text label. Exports true or false depending on whether the box is checked.

Child Tags

  • <label> — text displayed next to the check box
  • <default> — initial state (true or false)
  • <variable> — shell variable name for the value
  • <action> — command to run when the state changes

Signals

  • toggled (default) — fires when the user clicks the box

Example

<checkbox>
  <label>Enable logging</label>
  <variable>chkLog</variable>
  <default>true</default>
  <action>echo Logging: $chkLog</action>
</checkbox>
