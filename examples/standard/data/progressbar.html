<progressbar>

A horizontal bar that visualises progress from 0 to 100 percent. The value is typically fed from a shell command that prints numbers to stdout. The bar updates in real time as new values arrive.

Child Tags

  • <input> — a long-running command that prints progress values (0–100)
  • <variable> — shell variable for the current value
  • <label> — text displayed inside the bar
  • <action> — command to run on each value change

Attributes

  • width, height — bar dimensions in pixels

Example

<progressbar>
  <label>Copying files...</label>
  <input>for i in $(seq 0 5 100); do
    echo $i; sleep 0.2
  done</input>
  <variable>pbar</variable>
  <action>echo Done!</action>
</progressbar>
