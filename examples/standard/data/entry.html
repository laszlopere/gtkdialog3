<entry>

A single-line text input field. The user can type free-form text and the widget exports its contents as a shell variable when the dialog closes.

Child Tags

  • <default> — initial text shown in the field
  • <variable> — shell variable name for the value
  • <input> — shell command whose output sets the value
  • <action> — command to run when the user presses Enter

Attributes

  • visible — set to false to mask input (password mode)
  • editable — set to false to make the field read-only

Example

<entry>
  <default>localhost</default>
  <variable>entHostname</variable>
</entry>
