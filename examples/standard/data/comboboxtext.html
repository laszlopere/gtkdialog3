<comboboxtext>

A drop-down selector that lets the user pick one item from a list. Items can be provided statically with <item> tags or dynamically from a shell command.

Child Tags

  • <item> — a selectable option in the drop-down
  • <default> — the initially selected item text
  • <variable> — shell variable for the chosen value
  • <input> — shell command producing one item per line
  • <action> — command to run when the selection changes

Signals

  • changed (default) — fires when a new item is selected

Example

<comboboxtext>
  <item>English</item>
  <item>French</item>
  <item>German</item>
  <default>English</default>
  <variable>cmbLang</variable>
</comboboxtext>
