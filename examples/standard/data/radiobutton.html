<radiobutton>

A mutually exclusive toggle button. Radio buttons placed inside the same container automatically form a group — only one in the group can be active at a time. Each exports true or false.

Child Tags

  • <label> — text displayed next to the radio circle
  • <default> — initial state (true or false)
  • <variable> — shell variable for this button's state
  • <action> — command to run when selected

Signals

  • toggled (default) — fires when the selection changes

Example

<radiobutton>
  <label>Small</label>
  <variable>rbSmall</variable>
  <default>true</default>
</radiobutton>
<radiobutton>
  <label>Large</label>
  <variable>rbLarge</variable>
</radiobutton>
