<expander>

A collapsible container with a clickable header. When the user clicks the arrow the child widgets are shown or hidden. Useful for optional or advanced settings that should not clutter the main view.

Child Tags

  • <label> — text displayed in the expander header
  • <variable> — shell variable for the expanded state (true/false)
  • <action> — command to run when expanded or collapsed

Attributes

  • expanded — set to true to start in the open state

Example

<expander expanded="false">
  <label>Advanced Options</label>
  <vbox>
    <checkbox>
      <label>Enable debug logging</label>
      <variable>chkDebug</variable>
    </checkbox>
  </vbox>
  <variable>expAdvanced</variable>
</expander>
