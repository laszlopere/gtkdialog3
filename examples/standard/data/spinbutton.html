<spinbutton>

A numeric entry with up/down arrows for incrementing and decrementing the value. The user can also type a number directly. Exports the current numeric value as a shell variable.

Child Tags

  • <default> — initial numeric value
  • <variable> — shell variable for the current value
  • <input> — shell command whose output sets the value
  • <action> — command to run when the value changes

Attributes

  • range-min — minimum allowed value
  • range-max — maximum allowed value
  • range-step — increment per click (default: 1)
  • range-value — initial value (alternative to <default>)

Example

<spinbutton range-min="0" range-max="100"
            range-step="5" range-value="50">
  <variable>spnCount</variable>
</spinbutton>
