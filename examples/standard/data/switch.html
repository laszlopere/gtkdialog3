<switch>

A sliding on/off toggle, similar to a mobile-style switch. It exports true or false like a checkbox, but with a different visual style that clearly communicates an enabled/disabled state.

Child Tags

  • <default> — initial state (true or false)
  • <variable> — shell variable for the current state
  • <input> — shell command whose output sets the state
  • <action> — command to run when toggled

Signals

  • toggled (default) — fires when the switch is flipped

Example

<switch>
  <default>true</default>
  <variable>swEnable</variable>
  <action>echo Enabled: $swEnable</action>
</switch>
