<fontbutton>

A button that displays the currently chosen font and opens a font chooser dialog when clicked. Exports the selected font as a Pango font description string (e.g. Sans Bold 12).

Child Tags

  • <default> — initial font description
  • <variable> — shell variable for the chosen font
  • <input> — shell command whose output sets the font
  • <action> — command to run when a new font is picked

Signals

  • font-set (default) — fires when the user confirms a font

Example

<fontbutton>
  <default>Monospace 10</default>
  <variable>fntEditor</variable>
  <action>echo Font: $fntEditor</action>
</fontbutton>
