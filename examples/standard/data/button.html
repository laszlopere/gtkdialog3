<button>

A clickable push button. It can display a text label, a stock icon, or both. When clicked it runs its action commands. Stock buttons like ok and cancel automatically close the dialog.

Child Tags

  • <label> — text displayed on the button
  • <variable> — shell variable name for the button state
  • <action> — command to run when the button is clicked
  • <input file> — path to a custom icon image

Attributes

  • stock — use a stock button: ok, cancel, yes, no, etc.
  • image-position — icon placement: top, bottom, left, right

Example

<button>
  <label>Say Hello</label>
  <action>echo "Hello, world!"</action>
</button>

<button stock="ok"/>
