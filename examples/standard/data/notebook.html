<notebook>

A tabbed container that holds multiple pages. Each direct child widget becomes a separate tab. Tab labels are set with the <label> tag inside each page container.

Child Tags

  • <variable> — shell variable for the active tab index (0-based)
  • <action> — command to run when the user switches tabs

Attributes

  • tab-pos — tab position: top, bottom, left, or right
  • show-tabs — set to false to hide the tab bar

Example

<notebook tab-pos="top">
  <vbox>
    <text><label>First page content</label></text>
    <label>Page 1</label>
  </vbox>
  <vbox>
    <text><label>Second page content</label></text>
    <label>Page 2</label>
  </vbox>
  <variable>nbMain</variable>
</notebook>
