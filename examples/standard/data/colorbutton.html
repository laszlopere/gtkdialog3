<colorbutton>

A button that displays the currently chosen colour and opens a colour chooser dialog when clicked. Exports the selected colour as a hex string (e.g. #ff8800).

Child Tags

  • <default> — initial colour in hex (e.g. #ff0000)
  • <variable> — shell variable for the chosen colour
  • <input> — shell command whose output sets the colour
  • <action> — command to run when a new colour is picked

Signals

  • color-set (default) — fires when the user confirms a colour

Example

<colorbutton>
  <default>#3465a4</default>
  <variable>clrFg</variable>
  <action>echo Colour: $clrFg</action>
</colorbutton>
